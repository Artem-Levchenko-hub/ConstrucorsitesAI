## Hero = осознанный акцент: типографика-как-графика ИЛИ затемнённое фото/арт (prod @d4a900d)

**Запрос владельца:** на главном экране генерируемых сайтов сделать крутую графику — либо
крутая типографика, смахивающая на графику и делающая акцент, либо слегка затемнённое
фото/нарисованное вместо плоского фона. Просил sequential-thinking + ui-ux-pro-max + context7.

**Диагноз кита.** Кит уже богат на тип-помощники (`.display-fill`, `.gradient-text`,
`.text-shimmer`, `.split-chars`, `.line-rise`, `.kinetic-*`, `data-omnia-scramble`,
`data-anime="hero-stagger"`), НО не хватало двух самых ударных тип-как-графика приёмов
(контурный/полый шрифт и врезка текста в фото), и у арт-директора не было ОСОЗНАННОЙ
доктрины «герой = главный акцент» — hero был одним из визуалов, отсюда вялые экраны.

**Сделано (prompt + CSS, аддитивно):**
1. **Kit CSS** (все 4 static `omnia-kit.css`, блок v9.0): `.text-stroke`/`.text-stroke-2`/
   `.text-stroke-ghost` (полый шрифт, `-webkit-text-stroke`, `--stroke` дефолт currentColor)
   и `.text-blend`/`.text-blend-screen` (`mix-blend-mode` для врезки заголовка в кадр).
   ЧИСТАЯ статика, без `opacity:0` → capture-safe (урок reveal-бага): vision-гейт видит финал.
2. **Art-director** (`_ART_DIRECTOR_INSTRUCTION`): hero-правило переписано — обязателен
   выбор РОВНО ОДНОГО `ГЕРОЙ-РЕЖИМ`: I (ТИП-ГЕРОЙ) / II (ФОТО/АРТ-ГЕРОЙ; фото ИЛИ
   нарисованная графика, затемнение /45-/55, граф-слой+тон-грейд поверх). Вялый центр на
   плоском фоне = брак. Writer self-check #7 ловит режим.
3. **System prompt**: блок `_HERO_GRAPHIC` в `prompt_builder.py`, зарегистрирован в
   static-ветке после `_KIT_V6_REFERENCE`; переживает per-tier trim.

**Инструменты по просьбе:** ui-ux-pro-max (scrim 40-60%, no-emoji, контраст), context7
(Tailwind v4 `bg-clip-text`/arbitrary `[-webkit-text-stroke]`/v4.1 mask), sequential-thinking
(5 шагов: доктрина → capture-safety → точные CSS → формулировки AD → раскатка).

**Деплой.** gh-токен протух → push PAT'ом (owner подтвердил). Прод-compose
`apps/llm-gateway/deploy/full`: ff-merge → build api → up api+worker. Health: api+worker Up,
Uvicorn boot-complete, `_HERO_GRAPHIC`×2 и `text-stroke`×5 в образе, публичный
constructor.lead-generator.ru → 200.

**Скоуп/откат.** Только НОВЫЕ билды/rebuild (не surgical-edit, не существующие страницы),
только static-ветка (fullstack/spa без omnia-kit.css). Откат = `git revert d4a900d`.

**Правило.** Новые hero-эффекты-вход всегда floor-safe (статика = финальный вид, движение
только за `html.omnia-anim`+`no-preference`) — иначе headless-гейт меряет баг, не дизайн
(R-10 fail-soft, эхо reveal-бага).
