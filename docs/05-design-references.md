# 05. Дизайн-референсы и правила «крутого» сайта (для Claude Code)

> **Как пользоваться.** Открой этот файл, когда делаешь или ревьюишь дизайн сайта.
> Это библиотека вкуса: «приём → как повторить», дистиллированная из реальных топ-сайтов
> (Awwwards / Godly / студийные портфолио). Работает как общий гайд для любого фронта,
> не только Omnia.
>
> **Связь с кодом.** Операционные правила генератора Omnia — в
> [`04-generation-rules.md`](04-generation-rules.md) (промпт `prompt_builder.py` + кит
> `omnia-kit.{css,js}`). Здесь — расширенная карта вкуса и галерея референсов. Меняешь
> правило здесь — синхронизируй `_SIGNATURE_MOVES`/`_STYLE_KIT` в `prompt_builder.py`.

## 0. Главный принцип

**СДЕРЖАННОСТЬ + УВЕРЕННОСТЬ.** Монохром / почти-моно + ОДИН акцент, крупная уверенная
типографика, воздух как материал, движение по делу (не ради эффекта). Иерархия — масштабом
и пустотой, а не цветом и рамками. Цель — чтобы человек со вкусом сказал «сделано с душой»,
а не «очередной AI-лендинг».

## 1. Плейбуки по эстетикам

### 1.1 Скандинавский / нордический минимализм
**Когда:** эко, дом/мебель/интерьер, велнес, крафт, устойчивость, премиум-простота, B2B-«взрослый».

- **Палитра — «цвет из природы».** Тёплый off-white (`#F4F1EC` / `#F7F4EF`), песок/камень/
  глина/пыльный шалфей/приглушённый фьорд-синий, чёрный текст. БЕЗ ярких акцентов.
- **Сетка.** Строгая модульная (швейцарская школа): сильное выравнивание, асимметрия с балансом.
- **Типографика.** Нейтральный грайтеск (Inter, Hanken Grotesk, **Schibsted Grotesk** —
  норвежский, Geist) ± редакционный serif для контраста (Newsreader, Fraunces). Крупно, спокойно.
- **Воздух.** Огромные поля/паддинги; пустота — активный композиционный элемент («lagom»).
- **Фото.** Естественный свет, светлое, много воздуха вокруг; продукт / архитектура / природа.
- **Детали.** Hairline-разделители, ноль или едва теней, плоские поверхности, мелкие капс-лейблы.
- **Motion.** Медленные `.reveal`/fade, без спецэффектов.
- **Референсы:** carterco.us, studiosentempo.com (тип), pillandpillow.com (Swiss), бренды
  Vipp · Hay · Muuto · Frama · &Tradition · Acne Studios · Nordic Nest · Norse Projects;
  студии Kurppa Hosk · Bond (Helsinki) · Heydays / Bleed / ANTI (Oslo) · Stockholm Design Lab.
- **Пресет Omnia:** `NORDIC MINIMAL` (или `REFINED MINIMAL` для деловой версии).

### 1.2 Редакционный минимализм / тип как герой
**Когда:** студии, портфолио, манифесты, спецпроекты, афиши, культура, премиум-бренд.

- Гигантская типографика как контент; иерархия только масштабом; почти ноль хрома.
- Монохром + один резкий акцент (электрик-жёлтый/лайм/клевер) ИЛИ чёрно-белое.
- Вариативный шрифт с игрой осей (морф веса/ширины); редакционная/асимметричная сетка.
- Безрамочное чередование светлых/тёмных секций (ритм фоном, не линиями).
- **Референсы:** gt-flexa.com, k95.it / stelvio.k95.it, efesto.studio, brooksengland.com/scape,
  victionary.com, ortederbonnerrepublik.de.
- **Пресет Omnia:** `KINETIC TYPE`, `BOLD STUDIO`, `EDITORIAL LUXURY`.

### 1.3 Студийная графика / 3D-вау / motion
**Когда:** креативные студии, продуктовые спецпроекты, бренды с характером, «вау»-портфолио.

- Большая красивая графика несёт смысл; вёрстка вокруг неё — простая и чистая.
- **Дёшево (сейчас, без модели):** генерим графику кодом — глянцевые гради-сферы (`.blob`),
  gradient-mesh (`.bg-mesh`/`.bg-aurora`), `.grain` → «3D-ish» арт-герой за $0.
- **Дорого (roadmap):** AI-генерация картинок (text→image) для уникальных hero/иллюстраций —
  отдельная кросс-зонная фича (см. [`memory: omnia-graphics-strategy`]).
- Премиум смус-скролл (инерционный, Lenis по CDN) — «дорогое» ощущение; reduced-motion safe.
- **Референсы:** studiosentempo.com, polybion.bio, studiogusto.com / play2.studiogusto.com,
  bramnaus.com, studiofreight.com (Lenis), takeboost.com, adidaschile20.com, admoms.ru,
  possiblegroup.ru.

## 2. Сигнатурные приёмы (приём → как повторить → класс кита Omnia)

| Приём | Как повторить | Класс кита | Источник |
|---|---|---|---|
| **Тип как герой** | заголовок во весь экран, иерархия масштабом, оси вариативного шрифта | `.display-fill`, `.kinetic-type` | gt-flexa |
| **Архитектурный кроп** | фото в осмысленный силуэт (горы↔skyline) через `clip-path`, на плоском поле, заголовок внахлёст | `.clip-arch`, `.text-protect` | brooks/scape |

| **Тихие кейсы** | крупное фото-обложка с минимумом текста; лента доверия `.marquee` | `.img-zoom`+`.reveal`, `.marquee` | efesto, k95 |
| **Whitespace как материал** | одно заявление-заголовок, разреженная курируемая сетка | воздух + `.reveal` | k95 |
| **Глянцевая CSS-графика** | кластер морфящихся гради-сфер + grain = «3D-ish» арт-герой без картинок | `.blob`, `.bg-mesh`, `.grain` | sentempo, polybion |
| **Премиум смус-скролл** | инерционная прокрутка (Lenis CDN), reduced-motion off | Lenis CDN + init | studiofreight |
| **Крупные арт-фото** | одно большое художественно-кадрированное фото вместо россыпи мелких | `.img-zoom` | brooks, k95 |

## 4. Шрифты: коммерческий → бесплатный аналог

Генерация использует только **Google Fonts** и **Fontshare** (`api.fontshare.com`).

| Коммерческий (с референсов) | Бесплатный аналог |
|---|---|
| Aeonik · Neue Montreal · Neubau · Helvetica Neue | **Hanken Grotesk** · General Sans · Schibsted Grotesk · Geist · Inter |
| GT Flexa / GT Flexa Ext | **Bricolage Grotesque** · Anybody (вариативная ширина) · Unbounded |
| URW DIN | **Saira** · Oswald · Archivo |
| FS Kim · Perpetua · Stelvio | **Fraunces** · Instrument Serif · EB Garamond · Playfair Display |
| English Egyptian (slab) | **Roboto Slab** · Zilla Slab |

Правило вкуса: пара «выразительный дисплейный + чистый текстовый». Не ставь Inter рефлексом
«ради характера» — характер даёт дисплейный.

## 6. Чек-лист «не generic-AI»

- [ ] НЕ Inter/системный по умолчанию ради «характера» — характер даёт дисплейный шрифт.
- [ ] НЕ фиолетовый градиент на белом по умолчанию; цвет — приглушённый/сложный, один акцент.
- [ ] НЕ 4 одинаковые секции-карточки подряд; разнообразь ритм (bento, цитата, инверсия).
- [ ] НЕ центрируй всё; пробуй асимметрию, overlap, редакционную сетку.
- [ ] Два сайта подряд не на одно лицо: меняй шрифты, палитру, светлую/тёмную тему.
- [ ] Один оркестрованный page-load со staggered-появлением; движение по делу.
- [ ] Воздух щедрый; иерархия масштабом и пустотой, а не рамками везде.
- [ ] Атмосфера фоном (grain / mesh / orb / hairline), а не «просто заливка».

## 7. Как добавить новый референс (рабочий цикл)

1. **Сканируй визуально:** скриншот hero + вытащи computed шрифты/цвета
   (`getComputedStyle` через puppeteer/playwright), не только текст.
2. **Выдели ПРИЁМ** (что переносимо), а не копируй вёрстку/контент.
3. **Закодируй:** строка в `_SIGNATURE_MOVES` (`prompt_builder.py`) + при нужде новый класс
   в `omnia-kit.css` (4 копии шаблонов держать идентичными — сверять хэш). Целый вид → пресет
   в `_STYLE_KIT`.
4. **Задокументируй:** строка в галерее §3 здесь + правило в §1/§2; синхронизируй `docs/04`.
5. **Проверь рендер** в браузере (0 ошибок, reduced-motion, 375px).
