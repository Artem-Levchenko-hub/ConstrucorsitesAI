# 🗣️ Шпаргалка на созвон — один экран

## Открыть так
> «Не прототип на переписать — **рабочий двигатель**. Нужны ремни безопасности и пара проводов. Каждый знаем по имени, файлу и цене.»

## Гл. вопрос: «Яндекс Go» языковыми моделями?
- **Целиком — НЕТ.** Не наша слабость — **потолок технологии**, ни у кого.
- Почему: реал-тайм ядро, ML на данных, лицензии/платежи, мобайл в сторах, 99.99%, ответственность за деньги/жизни — это **люди-инженеры, ИИ ассистирует**.
- **ИИ-роль:** строит 60–80% рутины ВНУТРИ команды, ускоряет. Не заменяет организацию.

## Что строим УЖЕ сейчас (наш рынок)
- ✅ Кабинет клиники для **одной сети**: запись + роли + админка.
- ✅ Внутренние корп-системы (заявки, склад, расписание).
- ✅ Realtime-сервис (чат/лента) ограниченного масштаба.
- 🟡 Скоро: ранний SaaS, телеграм-бот с платежами, API.
- **Формула:** строим **ПРОДУКТ** (кабинет, модуль), не **ПЛАТФОРМУ** (масштаб, много клиентов, эксплуатация).

## Почему ещё не работает надёжно (самокритично)
**Часы — лекарство уже написано, не подключено:**
- инструмент «свежие доки» мёртв из-за **1 строки** → модель угадывает API;
- «прощающее» сравнение правок не подключено → правки срываются на дешёвой модели;
- усиление до сильной модели теряется на продолжениях.

**Дни — БЛОКЕР широкого запуска:**
- песочница не изолирована (общая сеть, открытый выход) → вектор утечки. Механизмы написаны, выключены.

**Недели — это уже инженерия:**
- серьёзные апп шипятся **без блокирующей проверки**; гейты ролей/безопасности — мёртвый код при стоящих галочках;
- нет изоляции данных между клиентами → «одна сеть» да, «коробка на сотни клиник» нет;
- мобайл / Go / Rust / .NET — нет вообще.

## Лестница «во что упирается»
```
ЧАСЫ    → docs(1 строка), прощающее сравнение, проброс усиления
ДНИ     → изоляция песочницы, api/бот в роутер
НЕДЕЛИ  → блокирующий гейт всем стекам, изоляция данных (RLS), новые стеки
МЕСЯЦЫ  → мобайл, эксплуатация/масштаб
ПОТОЛОК → «Яндекс Go целиком» — не ИИ, а команда инженеров
```

## Закрыть так
> «Двигатель работает и видит свои ошибки — сильнее конкурентов-одношотеров. До продакшна часть фиксов — часы, часть — недели, знаем каждый. Яндекс Go целиком не строит никто. А вертикали под ключ — кабинет клиники для сети, внутренние системы, ранний SaaS — наш рынок уже этого года.»
